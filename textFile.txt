 1. $ git touch textfile.txt__________________________________________|---> Создать в Main текстовый файл.
 2. $ nano texFile.txt________________________________________________|---> Добавить туда текст.
 3. $ git checkout -b newTB___________________________________________|---> Создать другую ветку.
 4. $ nano texFile.txt________________________________________________|---> Сделать изменения в текстовом файлев новой ветке.
 5. $ git add textFile.txt____________________________________________|---> Добавить файл.
 6. $ git commit -m "Added new textline, chose the first line"________|---> Закомитить изменения.
 7. $ git checkout master_____________________________________________|---> Перейти в Main (master).
 8. $ git merge newTB_________________________________________________|---> Вмержить новую ветку в Main (master).
 9. $ git push________________________________________________________|---> Обновлённый Main запушить во внешний репозиторий. 
10. $ git push origin newTB___________________________________________|---> Новую ветку запушить во внешний репозиториий.
